# Rent Calculator

A simple Python project to calculate rent, food expenses, and electricity bill shared among roommates.

## Features
- Calculate electricity bill
- Split total expenses equally
- Beginner-friendly Python project

## Technologies Used
- Python

## How to Run

```bash
python rent_calculator.py
```

## Example

Enter the rent of hostel/PG = 10000
Enter the amount of food ordered = 2000
Enter total number of units used per month = 150
Enter charge per unit = 8
Enter the number of persons living in room = 4

Each person will pay = 3300.0
